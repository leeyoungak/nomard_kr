# Persona
너는 지금부터 UI 전문가야. 현재 프로젝트의 시안을 4개 더 만들려고 해.

# 작업
아규먼트로 입력한 4가지 테마로 4개의 UI 시안을 제작해줘. 4개의 시안은 모두 독립적인 subagent를 생성해서 동시에 parallel하게 작업해줘.

Agent tool을 사용할 때 `isolation: "worktree"` 파라미터를 사용해서 각 subagent가 독립적인 worktree에서 작업하도록 해줘.

## 각각 subagent별 작업 방법
- Agent tool의 `isolation: "worktree"` 옵션을 사용해서 독립적인 환경에서 작업해줘
- 할당된 디자인 스타일로 UI를 변경해줘
- 시안을 볼 수 있도록 개발 서버를 시작해줘 (포트: 4001, 4002, 4003, 4004를 각 에이전트에 순서대로 할당)
- 만약에 에러가 있다면 시작될 때까지 수정해줘
- 각 subagent에게 사용할 포트 번호를 명시적으로 전달해줘

## 포트 할당
- 1번 subagent (첫 번째 테마): 포트 4001
- 2번 subagent (두 번째 테마): 포트 4002
- 3번 subagent (세 번째 테마): 포트 4003
- 4번 subagent (네 번째 테마): 포트 4004
